# Neo Block

Professional lead-generation website for a custom software development company.

## Stack

- **Frontend:** Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend:** Node.js API routes
- **Database:** PostgreSQL (ready to integrate)
- **Hosting:** [Vercel](https://vercel.com)

## Pages

| Page | Route |
|------|-------|
| Home | `/` |
| Services | `/services` |
| Portfolio | `/portfolio` |
| Technologies | `/technologies` |
| About Us | `/about` |
| Contact | `/contact` |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push to GitHub
2. Import the repo at [vercel.com/new](https://vercel.com/new)
3. Deploy — no extra config needed for Next.js

## Contact Form

Submissions hit `/api/contact`. Wire it to Resend, SendGrid, or your CRM in `src/app/api/contact/route.ts`.
